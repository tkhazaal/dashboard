# Logo

To use your **exact** logo image instead of the styled wordmark, save it here as:

    public/img/logo.png

(A transparent-background PNG works best — the dashboard auto-inverts black art to
white for the dark sidebar.) The sidebar shows this file automatically when present,
and falls back to the CSS "Tania Khazaal" wordmark when it's missing.

The browser tab icon is `public/favicon.svg` (a "TK" monogram).
